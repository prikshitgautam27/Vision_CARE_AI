# Vision_CARE_AI
VISIONCARE AI is a cataract detection ensemble web application built with Streamlit and Keras. It empowers clinics and healthcare providers to upload retinal fundus images, process them through an ensemble of deep learning models, and instantly generate diagnostic reports.
for dataset credits to KAGGLE DATASETS: "https://www.kaggle.com/datasets/andrewmvd/ocular-disease-recognition-odir5k"
